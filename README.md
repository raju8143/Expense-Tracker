## How to Use

1. **Clone the repository:**
   ```bash
   git clone https://github.com/raju8143/Expense-Tracker.git
   cd Expense-Tracker
   ```

2. **Open the application:**
   - Simply open `index.html` in your web browser
   - No installation or build process required

3. **Add a Transaction:**
   - Enter the date
   - Enter the amount
   - Select the transaction type (Income/Expense)
   - Click "Add" to record the transaction

4. **View Summary:**
   - Total Income, Total Expenses, and Balance are updated in real-time
